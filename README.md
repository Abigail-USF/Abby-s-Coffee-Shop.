# Abby's Coffee Shop

## Project Description

This project is a responsive small-business website for a fictional coffee shop. It presents the shop's services, contact information, and a customizable call-to-action headline that users can update using JavaScript.

## Technologies Used

- HTML
- CSS
- JavaScript
- DOM manipulation
- Event listeners

## How to Run the Project

1. Clone or download the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in a web browser.
4. Enter a new headline and select the Update button to view the JavaScript interaction.

## What I Learned

This project helped me practice building a structured webpage with HTML, styling it with CSS, and using JavaScript to interact with the DOM. I learned how to select page elements, respond to button clicks, and update text based on user input.
